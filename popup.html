How to Connect to Google Sheet & Apps Script
  1. Open your Google Sheet and go to Extensions > Apps Script.
  2. Paste the code (use the button below to copy) and save.
  3. Click the Deploy ⚙️ (gear) icon at the top right, then select Web app.
  4. Enter the description as Web-to-Sheet Logger.
  5. Set access to Anyone and deploy. Authorize if prompted.
  6. Copy the Web App URL (2nd URL ends with /exec) and add it above.
